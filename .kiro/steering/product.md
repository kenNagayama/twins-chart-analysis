# Product Overview

トロリ線摩耗検測データ（5cm ピッチ生データ）を対象とした探索的データ分析（EDA）ツール。
架線設備に取り付けられたトロリ線の残存直径を連続計測したチャートデータから、ノイズを除去しつつ局所的な異常摩耗を定量的に検知することを目的とする。

## Core Capabilities

- **データ読み込みと検証**: Excel ファイル（.xlsx）から計測データを読み込み、必須10列の存在を検証する
- **CH別インタラクティブ可視化**: 計測チャンネル（CH 1〜4）ごとに摩耗チャートをインタラクティブに表示し、日本語属性のホバーツールチップを提供する
- **ノイズフィルタリング**: 移動中央値フィルタおよびSavitzky-Golayフィルタを ON/OFF 切り替えで適用し、スパイクノイズを除去する
- **信号解析（EDA）**: スライドウィンドウによる RMS・FFT・STFT 処理でノイズ特性と最適パラメータを探索する
- **Z-Score 異常検知**: フィルタ後データの移動平均からの乖離（Z-Score）を算出し、局所的摩耗異常箇所を定量的に検出する

## Target Use Cases

- データアナリストが Streamlit Web アプリのサイドバーウィジェット（スライダー・トグル）でパラメータを対話的に調整しながら分析する
- 生波形・フィルタ後波形・Z-Score プロットを上下3段で連動比較し、各処理段階の効果を一目で評価する
- 分析結果を HTML ファイルとして出力・共有する

## Value Proposition

手動チャート確認作業を再現可能・自動化されたパイプライン処理に置き換え、ノイズとの混同による異常見落としリスクを低減する。
異なる路線・区間のデータにも同一パイプラインを適用できる拡張性を持つ。
